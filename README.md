# NFC_Honeypot
Know Your Enemy: A Proactive Security System That  Tracks WHO Attacks, WHAT They Steal, and WHEN It Happens

 OBJECTIVE 
To develop an intelligent NFC honeypot system that monitors 
unauthorized access attempts on contactless cards, tracks data 
theft in real-time, and generates forensic-grade attack reports 
for security incident analysis and threat intelligence.

KEY FEATURES 
✓ Real-time attack detection and logging
✓ Threat level classification (1-10 scale)
✓ Forensic-grade audit trails
✓ IST timestamp synchronization
✓ UID-based attacker fingerprinting
✓ Multi-sector data theft tracking
